# 待办事项管理

## 使用说明

使用 Obsidian 原生的 checkbox 管理待办事项：

```markdown
- [ ] 待办事项
- [x] 已完成
```

## 当前任务 (`current/`)

按主题分组的当前任务：
- [[current/learning-python-specs]] - Python 规范学习
- [[current/resource-collection]] - 资源收集任务
- [[current/learning-dev-methods]] - 开发方法学习
- [[current/learning-frameworks]] - 框架技术学习

## 待办事项池 (`backlog/`)

未来的待办事项，需要时整理到 `current/`

## 已完成 (`done/`)

已完成的任务归档

## 快速添加

在 [[01-Daily]] 的日记中快速添加待办事项，定期整理到 `current/`

## 统计

使用 Obsidian 的标签系统过滤待办事项：`#todo`
