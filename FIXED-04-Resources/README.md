# 资源收集

## 目录内容

### `docs/` - 文档链接
官方文档和参考手册

### `blogs/` - 博客文章
技术博客和学习笔记

### `videos/` - 视频教程
视频教程和课程

## 分类标签

使用标签进行资源分类：
- `#resource/doc` - 文档
- `#resource/blog` - 博客
- `#resource/video` - 视频
- `#resource/tutorial` - 教程

## 添加资源

在相应目录下创建 Markdown 文件，记录资源链接和笔记：
```markdown
---
title: 资源名称
tags: [resource/doc, python]
source: https://example.com
---

# 资源名称

[原始链接](https://example.com)

## 笔记

...
```
