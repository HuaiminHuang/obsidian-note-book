# 标签系统

使用 `#` 符号为笔记添加分类标签。

## 语法

```markdown
#标签
#嵌套/子标签
```

## 示例

```markdown
这是一条笔记 #重要
另一条笔记 #项目/A #项目/B
```

## 标签面板

在左侧标签面板可以看到所有标签及其使用频率，点击即可筛选。

## 搜索语法

```markdown
tag:#工作           包含该标签
tag:#项目/A         嵌套标签
-tag:#已完成        排除该标签
```

## 与普通 Markdown 的区别

标准 Markdown 中 `#` 仅用于标题，Obsidian 拓展为标签功能。
