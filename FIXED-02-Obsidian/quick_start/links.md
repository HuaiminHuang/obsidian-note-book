# 双向链接

Obsidian 最核心的特性，用 `[[ ]]` 语法创建笔记间的连接。

## 基础语法

```markdown
[[笔记名称]]           直接链接
[[笔记名称|显示文本]]   自定义显示文本
[[笔记名称#标题]]      链接到特定标题
[[笔记名称#标题|文本]]  链接+自定义文本
```

## 示例

- [[README]] - 链接到 README 笔记
- [[README|返回首页]] - 显示为"返回首页"

## 块引用

引用笔记中的特定段落：

```markdown
[[笔记名#标题]]        链接到标题
[[笔记名^block-id]]   链接到块（需先定义块ID）
```

定义块 ID：

```markdown
这是一个段落 ^my-block-id
```

## 反向链接

在笔记右侧面板可查看所有引用当前笔记的其他笔记，形成双向连接。

## 与标准 Markdown 的区别

| 标准 MD | Obsidian |
|---------|----------|
| `[链接](file.md)` | `[[file]]` |
| 相对路径 | 笔记名即可，自动解析 |
| 单向链接 | 双向（自动追踪引用） |
