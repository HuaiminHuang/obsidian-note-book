# Obsidian 快速入门

Obsidian 是一个基于本地 Markdown 的知识管理工具，相比普通 Markdown 有以下独特特性。

| 嵌入内容 | 引入其他笔记/图片 | [[embeds]] |

## 与标准 Markdown 的差异

| 功能 | 标准 Markdown | Obsidian |
|------|---------------|----------|
| 链接 | `[文本](路径)` | `[[笔记名]]` 或 `[[笔记名\|显示文本]]` |
| 图片 | `![](path)` | `![[image.png]]` |
| 标签 | 无 | `#标签` |
| 嵌入 | 无 | `![[笔记名]]` |
| 块引用 | 无 | `[[笔记名#标题]]` |

## 快捷键速查

| 操作 | Windows/Linux | macOS |
|------|---------------|-------|
| 快速切换 | `Ctrl+O` | `Cmd+O` |
| 命令面板 | `Ctrl+P` | `Cmd+P` |
| 新建笔记 | `Ctrl+N` | `Cmd+N` |
| 搜索 | `Ctrl+Shift+F` | `Cmd+Shift+F` |
| 切换预览 | `Ctrl+E` | `Cmd+E` |

## 常用操作

1. **创建链接**: 输入 `[[` 后会自动提示已有笔记
2. **查看反向链接**: 右侧面板显示哪些笔记引用了当前笔记
3. **关系图谱**: 查看笔记间的连接关系
4. **标签检索**: 点击标签或使用 `tag:#标签名` 搜索

---

> [!tip] 建议
> 使用双向链接构建你的知识网络，让笔记之间产生连接。
