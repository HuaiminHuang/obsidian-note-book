# 嵌入内容

使用 `!` 前缀嵌入其他笔记或媒体文件。

## 语法

```markdown
![[笔记名]]           嵌入整个笔记
![[笔记名#标题]]      嵌入特定标题下的内容
![[笔记名^block-id]]  嵌入特定块
![[图片.png]]         嵌入图片
![[视频.mp4]]         嵌入视频
```

## 嵌入笔记

```markdown
![[README]]
```

上面会直接显示 README 的内容，而不是链接。

## 嵌入部分内容

```markdown
![[links#基础语法]]   只嵌入"基础语法"部分
```

## 图片尺寸调整

```markdown
![[image.png]]           原始尺寸
![[image.png|300]]       宽度 300px
![[image.png|300x200]]   宽 300 高 200
```

## 与标准 Markdown 的区别

| 标准 MD | Obsidian |
|---------|----------|
| `![](path)` | `![[filename]]` |
| 仅图片 | 支持笔记、视频、音频、PDF |
| 无尺寸参数 | 支持 `\|尺寸` 参数 |
